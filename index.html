GridKid Spreadsheet Application

GridKid is a spreadsheet application that is designed to run in the terminal. I started working on it as part of my CS 430 class at JMU (Programming Languages) with Dr. Chris Johnson and am using Ruby to implement it. It's designed to present a table of cells that the user can fill with data and/or formulae to perform computational tasks, as you would with a spreadsheet application like Microsoft Excel, for example. The project is broken down into four separate milestones: Model of the Spreadsheet, an Interpreter, an Interface, and Variables and Control Flow.

Virtual Reality App

This is a virtual reality application that I created using Unity as a part of a virtual reality elective at JMU. The app is designed to simulate the dangerous effects of climate change if it continues to get worse. For example, the first scene is of a flooded city, the second is a powerful hurricane, the third is a forest fire, the fourth is the last iceberg melting, and the last scene is of a dried up lake due to severe drought. The user can explore each scene and interact with certain materials.

RISC-V ALU

This is a 8-bit RISC-V ALU that my partner, Joey Blethen, and I implemented for our Computer Architecture systems elective at JMU. We implemented it using the hardware description language Verilog and with the Vivado Design Suite. It takes two 8-bit operands, an ALUop input, and a function code from a instruction and it outputs an 8-bit result, along with a zero line indicating if the result is zero, an overflow line, and a carry out line.

Command - Line Shell

This is a command-line shell that I implemented as part of a Computer Systems class at JMU with Dr. Kirkpatrick. It was coded with C and operates similar to the bash shell in Linux. The current implementation of this shell is minimally functional and can run external programs and built-in commands, such as ls, head, and which. It also provides access to return codes.
