About Me

My name is Alex Karapetkov. I graduated with a Computer Science degree and minor in History from James Madison University in May 2024. This portfolio is meant to showcase some of the projects, both academic and personal, that I have completed in the last few years and some that I am still working on. They range from web applications made with HTML, CSS, and Javascript to the modeling of computer hardware, such as an ALU and program counter, using hardware description languages like Verilog.

Projects

GridKid Spreadsheet Application

GridKid is a spreadsheet application designed to run in the terminal. Developed as part of CS 430 at JMU, it uses Ruby to implement a table of cells for data and formulae, similar to Microsoft Excel. The project is broken into milestones: Model, Interpreter, Interface, and Variables/Control Flow.

Check it out!

Virtual Reality App

This VR app simulates climate change’s dangerous effects, including floods, hurricanes, forest fires, melting icebergs, and droughts. Built with Unity as a JMU elective project, users can explore and interact with each scene.

Check it out!

RISC-V ALU

This 8-bit RISC-V ALU, co-created with Joey Blethen for a Computer Architecture course, uses Verilog and Vivado. It processes operands and instruction codes to output results and status signals like zero, overflow, and carry out.

Check it out! Project Writeup

Command-Line Shell

A bash-like command-line shell implemented in C for a Computer Systems course. It supports external programs and built-in commands like ls, head, and which, including return codes.

Check it out!

Applied Algorithms Labs

Labs coded in Python for an Applied Algorithms class at JMU, covering recursion, backtracking, dynamic programming, greedy algorithms, Prim’s, Bellman-Ford, and Dijkstra’s algorithms.
